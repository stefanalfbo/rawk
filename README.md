# rawk

![Build and test](https://github.com/stefanalfbo/rawk/actions/workflows/build_and_test.yml/badge.svg)

A Rust implementation of AWK with the goal of achieving POSIX compatibility. This project aims to reimplement the AWK programming language in Rust while adhering to the POSIX specification.
